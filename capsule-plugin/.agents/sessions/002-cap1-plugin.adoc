== Ce qui a été fait

=== Dépendance compileOnly slider-gradle

=== CapsuleModels.kt — Modèles de données
- `SlideSegment(index, title, speakerNote)` : une slide
- `CapsuleScript(deckName, slides)` : un deck complet
- `CapsuleExtension` : DSL utilisateur avec propriétés configurables

  - `ttsVoice` (default: fr_FR-siwis-medium)
  - `outputDir` (default: build/capsule)
  - `sliderScriptDir` (default: build/capsule)
  - `viewportWidth/Height` (default: 1408x792)

=== CapsuleManager.kt — Gestionnaire de tâches
- `CapsuleScriptTask` : lit *-script.txt de slider-gradle, parse, valide
- `CapsuleBuildTask` : dépend de generateCapsuleScript, génère TTS placeholder par slide
- `CapsuleManager` : point d'entrée, registre les tâches via Project extension

=== CapsulePlugin.kt — Point d'entrée Gradle
- Crée l'extension `capsule`
- Délègue l'enregistrement des tâches à CapsuleManager
