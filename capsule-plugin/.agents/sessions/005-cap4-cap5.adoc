== Réalisations

=== CAP-4 — CapsuleVideoTask orchestration

. *Injection audio fallback séquentiel* (`CapsuleManager.kt:injectAudioSequentialFallback`) :
  ajout d'un parseur regex qui assigne séquentiellement les audios aux `<section>`
  quand `data-capsule-slide` n'est pas présent dans le HTML.
. *Multi-deck* : le code boucle déjà sur tous les scripts, donc les 2 decks sont
  traités indépendamment. Test vérifie que 2 `.webm` distincts sont produits.
. *Test @Tag("integration")* : exclu du `test` standard via `excludeTags("integration")`,
  tâche `integrationTest` dédiée. Si chromium dispo, capture réelle avec Playwright +
  validation taille > 0.
. *NoOp TTS ↔ NoOp Playwright bridge* : `resolvePlaywrightCapture()` détecte
  `ttsEngine == "noop"` et force `NoOpPlaywrightCapture()` pour éviter timeout
  sur `audio-X.ended`.

=== CAP-5 — Cucumber BDD

. *Dépendances* : cucumber-java 7.34.3, cucumber-junit-platform-engine,
  cucumber-picocontainer, junit-platform-suite 1.14.3. Bundle `cucumber` dans
  libs.versions.toml.
. *CucumberTestRunner* : @Suite + @IncludeEngines("cucumber") + configuration
  annotations (GLUE, FEATURES, PLUGIN, FILTER_TAGS).
. *5 scenarios Gherkin* (`src/test/features/capsule.feature`) :
  - Generate a WebM capsule from a deck with capsule script
  - NoOp fallback when Chromium is unavailable
  - Sequential fallback for deck without data-capsule-slide attributes
  - Multi-deck build produces separate videos
  - Audio injection into deck HTML
. *Step Definitions* : `CapsuleStepDefinitions.kt` avec GradleRunner + fixtures
  vers les chemins conventionnels (`build/docs/asciidocRevealJs/`,
  `build/capsule/`).
. *cucumberTest task* : classpath complet (main + test + jar), `excludeEngines("junit-jupiter")`,
  dépend de `classes`. Intégré dans `check`.

=== Corrections conventions CapsuleExtension

Les conventions contenaient `build/` en préfixe (`build/capsule`, `build/docs/...`)
alors que les résolveurs les préfixent déjà avec `buildDirectory.dir()`.
Corrigé en `capsule` et `docs/asciidocRevealJs`.
